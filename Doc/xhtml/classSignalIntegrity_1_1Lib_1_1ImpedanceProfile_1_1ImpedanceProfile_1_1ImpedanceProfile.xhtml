◆ rhoLimit

rhoLimit = 0.99
static

static float upper limit on absolute value of rho before the impedance profile measurement stops. it defaults to 0.99 it stops at this limit because this is effectively an open or short and continuing would be fruitless and cause the algorithm to blow up.
