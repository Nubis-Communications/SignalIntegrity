51  @param dfm float dissipation factor (loss-tangent) of mutual capacitance (F)
52  @param Gm float mutual conductance (S)
53  @param Lm float mutual inductance (H)
54  @param Z0 (optional) float reference impedance (defaults to 50 Ohms)
55  @param K (optional) integer number of sections (defaults to 0)
56  @note If K=0 is specified, it is modified to a value that will provided a very good numerical
57  approximation.
58 
59  The calculation is such that round-trip propagation time (twice the electrical length)

61  """
62  if K==0:
63  """max possible electrical length"""
64  Td=math.sqrt((max(Lp,Ln)+Lm)*(max(Cp,Cn)+2*Cm))
65  Rt=0.45/f[-1] # fastest risetime
66  """sections such that fraction of risetime less than round trip electrical
67  length of one section"""
68  K=int(math.ceil(Td*2/(Rt*self.rtFraction)))
