Detailed Description

Parses netlists and produces system descriptions.

These are instances of class SystemDescription.

This class provides a mechanism for producing system descriptions from netlists rather than by scripting the adding of devices, ports, device connections, etc.

Remarks
the dictionary of known devices is such that the key value looks like a netlist line following the device keyword and the reference designator. The value is the s-parameters. This addition of known devices allows weird devices to be defined by their s-parameters, but more importantly, allows the parser to be used for solutions where a file device does not need to be in a file.
