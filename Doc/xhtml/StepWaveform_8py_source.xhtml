29 
30  constructs a step waveform.
31 
32  @param td instance of class TimeDescriptor containing time axis of waveform.
33  @param Amplitude (optional) float amplitude of step (defaults to unity).
34  @param StartTime (optional) float starting time of the pulse (defaults to zero).
35  @param risetime (optional) float risetime in seconds (defaults to 0.)
36 
37  @note The amplitude can be positive or negative, with negative providing a negative
38  pulse.
39  @note The step starts at the first sample point after the start time specified.
40  @note actual risetime used is the sample period if less than that is specified
41  @note the risetime is applied such that the step reaches half amplitude at the start
42  time specified. Please note the expected non-causality.
43  """
44  x=[0 if t < StartTime else Amplitude for t in td.Times()]
45  T=risetime/self.rtvsT
46  rcStart=max(0,td.IndexOfTime(StartTime-T/2.))
47  if td.TimeOfPoint(rcStart)<StartTime-T/2: rcStart=min(rcStart+1,len(td)-1)
48  rcEnd=min(len(td)-1,td.IndexOfTime(StartTime+T/2.))
49  if td.TimeOfPoint(rcEnd)>StartTime+T/2: rcEnd=max(rcEnd-1,0)
50  for i in range(rcStart,rcEnd+1):
51  try:
52  x[i]=Amplitude*\
53  (math.sin((td.TimeOfPoint(i)-StartTime)/T*math.pi)+1.)/2.
54  except ZeroDivisionError:
55  pass
