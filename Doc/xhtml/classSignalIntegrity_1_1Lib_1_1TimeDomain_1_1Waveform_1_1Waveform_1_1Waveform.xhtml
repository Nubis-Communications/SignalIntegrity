def __mul__ (   self,
  other

overloads *

Parameters
otherinstance of class WaveformProcessor or float, int, complex to multiply by.
Returns
instance of class Waveform with other multiplied by self
Note
does not affect self
Waveform multiplication is an abstraction in some cases. The result for types of other is:
  • WaveformProcessor - returns self processed by the instance of WaveformProcessor.
  • float,int,complex - returns the Waveform produced by multiplying all of the values in self multiplied by the constant value supplied.
The most obvious type of WaveformProcessor is a FirFilter, but there are others like WaveformTrimmer and WaveformDecimator.
Exceptions
SignalIntegrityExceptionWaveformif other cannot be multiplied.

Parameters

reads a waveform from a file

Parameters
fileNamestring name of file to read
Returns
self
Note
this DOES affect self
the normal waveform format is one number per line starting with the horizontal offset followed by the number of points followed by the sample rate. The remaining lines contain one waveform point per line. This is the format output by SignalIntegrity. However, if the data is all on one line, then the format is assumed to be LeCroy MathPack format with the first point being the number of points, and the remaining points being time and value.
