59  """creates a symbolic matrix
60 
61  The matrix is of a simple form. It is a list of list of strings where the
62  string is either:
63  - rows=columns=1 - [[name]]
64  - rows <= 9 and columns <= 9 - list of list of name+'_'+str(r+1)+str(c+1)
65  @note if a row or column goes above 9, a comma is inserted between the port numbers
66  to remove ambiguity.
67  @param Name string name of device
68  @param Rows integer number of rows in the matrix
69  @param Columns (optional) integer number of columns in the matrix (defaults
70  to the number of rows).
71  """
72  if Columns == -1:
73  Columns = Rows
74  if Rows == 1 and Columns == 1:
75  return [[Name]]
76  return [[Name+'_'+(str(r+1)+str(c+1) if r<9 and c<9 else str(r+1)+','+str(c+1))
77  for c in range(Columns)] for r in range(Rows)]
