25  """produces calibrated or uncalibrated s-parameters from a network analyzer.
26  The usual use of this class is to provide calibrated s-parameters, given raw s-parameters

29  """Constructor
30  Calculates calibrated or raw s-parameters from a network analyzer.
31  @param f list of frequencies to calculate s-parameters
32  @param filename string filename containing (usually raw) s-parameters.
33  @param etfilename string filename containing error-terms (see Calibration).
34  @param portListstring (optional, defaults to None) string containing comma delimited list
35  of one-based port numbers to use. This is so that the error-terms from a large port count calibration
36  can be used to make a smaller measurement on less ports, and for port reordering. Essentially, this
37  is the list of network analyzer ports used for the measurement.
38  @param calculate (optional, defaults to True) bool that defines the calculation direction. If calculate
39  is True, then the assumption is that the s-parameters provided are raw and the goal is to convert the
40  raw s-parameters to calibrated s-parameters. In the unusual situation where calculate is False, the direction
41  is reversed and the assumption is that the s-parameters provided are calibrated and the goal is to convert the
42  calibrated s-parameters to raw s-parameters.
43  """
44  if portListstring!=None:
45  portlist=[int(p)-1 for p in portListstring.split(',')]
46  else: portlist=None

49  fixtures=calibration.Fixtures()

51  calibration.InitializeFromFixtures(fixtures)
