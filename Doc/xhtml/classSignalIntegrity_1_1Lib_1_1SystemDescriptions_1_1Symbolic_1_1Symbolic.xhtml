args(optional) named args (name=value) named arguments understood are:
  • 'size' - 'normal' or 'small' (defaults to 'normal')
  • 'docstart' - string for start of LaTeX document (defaults to a barebones article class document start.
  • 'docend' - string for end of document (defaults to normal LaTeX document end).
  • 'eqprefix' - prefix for equation start (defaults to \[).
  • 'eqsuffix' - suffix for equation end (defaults to \]).
  • 'eqenv' - boolean whether to enclose the equation in the equation environment Here, the size refers to the sizes of the output matrices. 'normal' means the normal size matrices and 'small' means the \smallmatrix environment.
