63  """symbolic three port current amplifier
64  @param G string current gain
65  @param Zi string input impedance
66  @param Zo string output impedance
67  @return list of list of string s-parameter matrix

71  """
72  D='3\\cdot Z0^2+ (2\\cdot '+Zo+'+2\\cdot '+Zi+'-'+G+'\\cdot '+Zo+' )\\cdot Z0+'+Zo+'\\cdot '+Zi
73  S11=lfrac(Zo+'\\cdot '+Zi+'+Z0\\cdot (2\\cdot '+Zi+'-'+G+'\\cdot '+Zo+' )-Z0^2',D)
74  S12=lfrac('2\\cdot Z0^2',D)
75  S13=lfrac('2\\cdot Z0^2+2\\cdot '+Zo+'\\cdot Z0',D)
76  S21=lfrac('2\\cdot Z0^2+2\\cdot '+G+'\\cdot '+Zo+'\\cdot Z0',D)
77  S22=lfrac(Zo+'\\cdot '+Zi+'+Z0\\cdot (2\\cdot '+Zo+'-'+G+'\\cdot '+Zo+' )-Z0^2',D)

80  S32=lfrac('2\\cdot Z0^2+2\\cdot '+Zi+'\\cdot Z0',D)
81  S33=lfrac(Zo+'\\cdot '+Zi+'-Z0^2+'+G+'\\cdot '+Zo+'\\cdot Z0',D)
