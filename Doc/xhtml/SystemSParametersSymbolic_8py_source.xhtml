44  @param args named arguments (name=value).
45  @return self
46  the named arguments possible are:
47  - 'solvetype' - (optional) either 'block' for the block solution or 'direct' for
48  the direct solution (defaults to 'block').
49  - 'size' - either 'normal', 'big', or 'biggest' (defaults to 'normal').
50  @see Symbolic
51  Here, 'size' is used depending on the size of the solution.
52  If the size is normal, then the block solution is output in matrix form
53  on one line. If the size is big, the inverse of (I-Wxx) is shown on one line
54  and Wxx is used on the next single line solution, as this tends to become
55  large sometimes. The biggest size is used when the solution is really large
