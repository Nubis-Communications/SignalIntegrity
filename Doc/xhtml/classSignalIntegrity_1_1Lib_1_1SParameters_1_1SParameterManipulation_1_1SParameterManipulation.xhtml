Parameters
epsilon(optional, defaults to 1e-6) absolute threshold on impulse response.
allLengths(optional, defaults to False) whether to return the lengths of each port combination
Returns
returns a tuple containining (negativeTime,positiveTime) if allLengths is false (default) containing the most negative time and most positive time for all impulse responses, otherwise if allLengths is true, it returns a list of list of tuples as stated for each impulse response.

Enforces both reciprocity, passivity and causality on the s-parameters.


Affects self.
First enforces reciprocity, then...
for up to the maxIterations specified, alternately enforces passivity, to the maxSingularValue and enforces causality (because each causes violations of the other, but alternating this way generally converges. If causality is detected by being below the causalityThreshold, then iterations stop.

Parameters
maxSingularValue(optional, defaults to 1) float maximumum singular value allowed
maxIterations(optional, defaults to 30) maximum iterations
causalityThreshold(optional, defaults to 0) positive float threshold for causality detection

  threshold = 0.

Checks whether the s-parameters are causal.

Parameters
threshold(optional, defaults to 0) positive float threshold for causality detection
Returns
boolean True if the absolute value of all values in the impulse response of each s-parameter before time zero are less than the threshold provided otherwise returns False.

This is checked by generating the impulse response corresponding to the frequency response of each s-parameter from and to port combination.
