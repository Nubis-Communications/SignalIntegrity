Converts power-wave s-parameters to pseudo-wave s-parameters.

Parameters
Splist of list power-wave based s-parameter matrix to convert
Z0w(optional) the reference impedance of the pseudo-wave based s-parameters (assumed 50 ohms)
Z0p(optional) the reference impedance of the power-wave based s-parameters (assumed 50 ohms)
Kw(optional) the scaling factor of the pseudo-wave based s-parameters (assumed to be sqrt(Z0w)
Returns
the converted s-parameters in pseudo-waves
See also
Z0KHelper to see how the reference impedance and scaling factor are determined for pseudo-waves.
Z0KHelperPW to see the reference impedance and scaling factor are determined for power waves.
